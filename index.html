Документация

Этот инструмент позволяет конвертировать файлы между различными форматами:

  • Конвертация изображений BTX ↔ PNG
  • Конвертация моделей DFF → MOD

Также включает инструмент для выбора цвета с выводом HEX кода.

Все преобразования выполняются локально в вашем браузере - файлы не загружаются на сервер.

Для пакетной обработки можно выбрать несколько файлов или использовать ZIP архивы.

Инструмент имеет минималистичный интерфейс и оптимизирован для быстрой работы.
